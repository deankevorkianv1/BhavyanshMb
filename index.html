About

Researcher with interest in Computer Networking, Decentralized Software Systems, Deep Reinforcement Learning, Robotics, General Artificial Intelligence, DevOps and Cloud Computing
In the past, I have researched on DevOps practices and Distributed Software Architectures. I currently have three Computer Science research publications at Peer-reviewed Conferences and Journals. Several research projects are still in progress.

I totally cherish Robotics projects and specifically those that employ Deep Learning in some form. I have worked on fixed-wing UAVs and generated two-dimensional Deep Q-Network based solutions for obstacle avoidance and path-planning system used by the UAV. My final degree Capstone project involved building a fully autonomous quadcopter with LIDAR-based SLAM and RRT-based Path-planning capabilities that could be used to map an unknown indoor environment.

Portfolio


Research

Security for Devops Deployment Processes: Defenses, Risks, Research Directions

A security analysis of DevOps tool chains at multiple levels in Cloud-based Software deployments on three different Case Studies. Tools used include Docker, Chef, Glassfish, AWS EC2, AWS CodeDeploy, AWS CloudFormation and Bash


CDEP: Continuous Delivery Educational Pipeline

PHP and Java Continuous Delivery Pipelines implemented using Docker, Jenkins, Maven, JUnit, PHPUnit, etc. to facilitate education in Continuous Delivery.


A Pilot Study on Introducing Continuous Integration and Delivery into Undergraduate Software Engineering Courses

Evaluating the Continuous Delivery Educational Pipeline (CDEP) by introducing the previously implemented pipeline in the Software Engineering Management course at the University of West Florida.

Robotics

Autonomous Drone Racing (FlightGoggles)

PID based drone racing controller with visual state estimation using C++(OpenCV) and ROS. Scored 17th Rank overall in AlphaPilot Challenge hosted by Lockheed Martin, MIT and Drone Racing League.






Indoor Mapping and Surveillance Quadcopter

Quadcopter capable of indoor mapping and localization using 2D SLAM and Altitude Hold flight. The quadrotor uses Pixhawk from 3DRobotics which employs the PX4 flight control stack.






Simultaneous Localization and Mapping (SLAM)

Stereo-based SLAM algorithms implementations: RTAB-Map, ORB-SLAM2, LSD-SLAM, Dense-SPTAM. The following video only shows the SLAM implementation based on RTAB-Map.






Planar Region Detection in 3D Point Cloud

Superfast Planar Region Segmentation and Plane Fitting using RANSAC implemented using the Point Cloud Library (PCL) in ROS. Marvel at the speed with which it refreshes and creates a brand new set of planes through the Point Cloud generated by RTAB-Map SLAM algorithm.






Rapidly-exploring Random Tree (RRT)

The JavaFX graphical interface that I had built for the Unmanned Systems Lab at UWF shows the Path-planning algorithm implemented using Randomly-exploring Random Tree (RRT) for Fixed-Wing UAVs. The generated path easily avoids the circular obstacles as shown in the image.






Android

Discoverers

Discoverers is a collaborative multiplayer game and a community of learners that enjoy casual gaming and those that want to learn more about the World. The players learn together by creating, sharing and solving challenges with one-another. The challenges are reviewed by the developers of the app before they are placed into the pool of quality challenges to solve.


The app allows the players to create, share, delete and review challenges with an intuitive User Interface. The game uses the Google Places API to allow the players to choose a location from anywhere on the Earth to either create or solve a challenge.

Differential-Drive Wheeled Robot (Bluetooth-Controlled)

This was one of my first robots. Built on the Arduino platform with bluetooth connectivity provided by the HC-05 module, this tiny robot could be controlled using an Android device. I was merely trying to follow an instructable here. Here's the link: Instructable



Parallel Port Controller

This project was my attempt to utilize the obsolete Parallel Port on my old desktop computer. A pendrive bootable DOS system was used to program the Parallel Port to control a 3-digit seven-segment display. Using a desktop computer as a microcontroller. Take a look! I also documented the entire project as an Instructable myself. Here's the link: Instructable
